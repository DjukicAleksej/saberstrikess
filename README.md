# Tempo Strike

A browser-based rhythm game controlled by hand movements using your webcam. Slash the sparks to the beat!

## Features
- **Motion Control**: Play using your hands via webcam (powered by MediaPipe).
- **Custom Music**: Upload your own MP3s to generate charts.
- **Modes**: Challenge yourself or relax in Zen Mode.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Enable camera access when prompted and step back to play!
